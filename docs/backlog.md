# Project Backlog

<div align="justify">&emsp;&emsp;
This document describes the product backlog, which is an ordered list of user stories, features, enhancements, and fixes that represent the work to be done for the project. It serves as a single source of truth for all requirements, helping the team prioritize development, plan sprints, and track progress toward delivering value to users.
</div>

## User Story #1

**Title:** Show time

> **As a** regular user, **I want** to be able to see the time on my watch, **so that** I maintain awareness of time throughout the day.

**Acceptance Criteria:**
- The watch shall display the current time in a human-readable format.
    - The watch must show both the hour, the minutes and seconds of the day on screen.
- The system shall set the time automatically through WiFi.
- The system shall let the user set the time manually.

**Status:** To Do  
**Type:** Feature

## User Story #4

**Title:** Play Bad Apple

> **As a** nerdy user, **I want** it to play the videoclip BadApple, **so that** i can be happy knowing it can play BadApple.

**Acceptance Criteria:**

## User Story #5

**Title:** Battery life  
> **As a** worker user, **I want** the battery to last longer than 8 hours, **so that** I can actually wear my CrapWatch throughout my whole shift.

**Acceptance Criteria:**
- The battery shall last at least 10 hours under normal use.
- Battery life must be displayed on screen.
- To save power, the screen must turn off while the watch is not being used.

## User Story #11

**Title:** Dice Roller  

> **As a** tabletop gamer, **I want** a dice roller on my CrapWatch, **so that** I can play RPGs without carrying actual dice.  

**Acceptance Criteria:**  
- The system shall provide a dice roller supporting common dice types (d4, d6, d8, d10, d12, d20, d100).  
- The result of the roll shall be displayed clearly on the screen.  
- Users shall be able to select which die to roll via button navigation.  
- The system shall generate random results using the ESP32 random number generator.
